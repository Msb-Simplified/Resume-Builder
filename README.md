# Resume-Builder
 Resume Builder is Laravel framework-based dynamic resume maker website. After login user can create his own RESUME with his own requirements. Users can download the Resume also send generated Resume.pdf via email.

1. `php artisan make:mail MyTestMail`
2. Edit `.env` file
```
MAIL_MAILER=smtp
MAIL_HOST=smtp.gmail.com
MAIL_PORT=587
MAIL_USERNAME=shishirbhuiyan83@gmail.com
MAIL_PASSWORD=appPassword
MAIL_ENCRYPTION=tls
MAIL_FROM_ADDRESS=shishirbhuiyan83@gmail.com
MAIL_FROM_NAME=Shishir Resume
```
